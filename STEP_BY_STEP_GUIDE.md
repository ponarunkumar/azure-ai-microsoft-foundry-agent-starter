## Getting Started

1. **Start in Microsoft Foundry** - Create and test your agent in the visual interface
2. **Export your configuration** - Note your project endpoint and agent name
3. **Use this template** - Clone and configure with your settings
4. **Expand programmatically** - Build on the foundation provided

This approach ensures you have a working agent before moving to code, reducing development friction and increasing success rates.
